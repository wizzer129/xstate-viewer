# Styles

This directory is for configuring the styles of the application.

- Use `main.less` as the global style entry imported by the app.
- Apply cross-browser/cross-OS resets and normalizations in `normalize.less`.
- Define design tokens in `variables.less`.
- Add reusable style helpers in `mixins.less`.
- Keep app-wide base element styling in `theme.less`.
- Keep `settings.scss` for Vuetify SASS variable configuration.
